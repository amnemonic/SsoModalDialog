# SsoModalDialog
Universal SSO Modal window to use in your application

# Usage

`SsoModalDialog.exe <SSO Url Address> [<SearchString>] [<SSO Title>]`

* `<Se<archString>` defaults to `SAMLResponse`
* `<SSO Title>` defaults to `Login`
